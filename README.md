# schnow265/nvim.conf

My extremey bad nvim config for my purposes.

## Usage

Add to `.chezmoiexternal.toml`:

```toml
[".config/nvim"]
type = "git-repo"
url = "https://gitlab.com/schnow265/nvim.conf.git"
refreshPeriod = "24h"
```

... and then run `chezmoi apply`
